class paretofront.ParetoFront(points, x_reverse, y_reverse)[source]¶

Bases: object

Computes nested Pareto Frontiers of points in 2d.

Calculates the Convex Hull of the set of points using qhull, then computes the Pareto Front of those points.

build_nextfront()[source]¶

Build the Pareto front of the points with the last front removed

If the next front is successfully built return True, otherwise False. Since points array does not have duplicates, the pruned points array does not have duplicates.
