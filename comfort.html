comfort module¶

Original Developer: David Roberts Purpose of Module: To evaluate passenger comfort using

Sperling’s comfort index (Wz).

Last Modified: 7/16/15 Last Modified By: Jonathan Ward Last Modification Purpose: To clarify naming and add citations.

Citations:
“Ride comfort in Tilting Trains”
  • Johan Forstberg
“Experimental and Analytical Ride Comfort Evaluation of a Railway Coach”
      1. Gangadharan

OUTLINE of SCRIPT (comfort.py): input vars: {t_i}, {x_i}, {v_i}, {a_i} (four lists of points in 3_d space) output vars: c (comfort rating)

SUB-SCRIPTS:
  1. Find acceleration in passenger frame:

    {t_i}, {v_i}, {a_i} –> {ap_i}

  2. Take Fast Fourier Transform of passenger accel:

    {ap_i} –> {ap_f}

  3. Apply weighting filter and sum results

    {ap_f} –> {w(f)*ap_f} –> c = sum_f |w(f)*ap_f|^2
